# MSA-GEM-CCB
Esta plataforma foi desenvolvida com base no conteúdo apresentado no novo Método Simplificado de Aprendizagem Musical da Congregação Cristã no Brasil. Não temos a intenção de substituir os materiais disponibilizados pela CCB. Nosso objetivo é desenvolver e disponibilizar funcionalidades que auxiliem e complementem o estudo musical.
